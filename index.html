ระบบจัดการสินค้าตัวอย่าง — แก้ไข / ดู / ประวัติ

หน้าเว็บ

  • Catalog (หน้าแสดงสินค้าสำหรับลูกค้า)
  • History (ประวัติรับเข้า/ตัดออก)
  • Admin (แผงผู้ดูแล)

วิธีทดสอบเบื้องต้น

  1. เปิด config.js แล้วตั้งค่า APPS_SCRIPT_URL เป็นเว็บแอป Apps Script ที่คุณ deploy
  2. เปิด admin.html เพื่อทดสอบเพิ่ม/แก้ไข/ลบ
  3. เปิด catalog.html เพื่อดูรายการสินค้า (ใช้ action=list)
